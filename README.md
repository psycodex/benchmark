# MikroORM 4 vs TypeORM benchmark

This benchmark is using SQLite 3, comparing CRUD operations on 10 000 entities.
To use other drivers, run `docker-compose up` first or adjust the configuration
in `perf.ts` file. 

## Running

```sh
# for mikro-orm 
cd mikro-orm
yarn
yarn start sqlite
yarn start mysql
yarn start mariadb
yarn start pg
yarn start mongo
cd ..

# for typeorm 
cd typeorm
yarn
yarn start
cd ..
```
